### Chatty BWA


## 🛠 Technologies Used

* Flutter
* Dart

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/obitwicaksono/chatty-bwa-flutter.git
```

2. Navigate to the project directory:

```bash
cd chatty-bwa-flutter
```

3. Install the dependencies:

```bash
flutter pub get
```
